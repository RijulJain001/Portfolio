# Rijul Jain Portfolio

This is a static portfolio site for `Rijul Jain`, built with HTML, CSS, and JavaScript.

## What is included

- `index.html` — main portfolio homepage
- `login.html` — optional secure access portal styled page
- `styles.css` / `login.css` — page styles
- `script.js` / `login.js` — interactive behavior
- `assets/` — profile placeholder, resume PDF, and supporting assets

## Recommended repo setup for GitHub Pages

1. Create a new GitHub repository, for example `portfolio`.
2. Add these files and commit the project.
3. Push to the `main` branch.
4. Enable GitHub Pages from repository settings.
   - Source: `GitHub Actions` or `main` branch root.

## Automatic deployment with GitHub Actions

A GitHub Actions workflow is included at `.github/workflows/pages.yml`.
When you push to `main`, GitHub Pages will automatically deploy the repository contents.

## How to preview locally

Open `index.html` in your browser to view the portfolio.
